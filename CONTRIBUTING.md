# Contributing

Thanks for your interest in contributing. This is a small portfolio project, so contributions are lightweight and focused on bug fixes or small improvements.

## How to Contribute

1. Fork the repo and create a feature branch.
2. Keep changes scoped and well documented.
3. Run tests if you change behavior:

```bash
d:/f1RacePred/.venv/Scripts/python.exe scripts/smoke_test.py
```

4. Open a pull request with a short summary and screenshots if UI changes are included.

## Code Style

- Keep code clear and minimal.
- Avoid large refactors unless they are necessary.
- Prefer explicit configuration via env vars.
